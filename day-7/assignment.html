Introducation of CSS

CSS (Cascading Style Sheets) is a styling language used to design and format web pages.
It helps developers control colors, fonts, layouts, spacing, and overall appearance
of HTML elements, making websites more attractive and user-friendly.

CSS separates the design from the content of a web page, which makes websites easier to
maintain and update. By using CSS, developers can apply consistent styles across multiple
pages, improving both efficiency and user experience.

Modern CSS provides powerful features such as Flexbox, Grid, animations, and responsive design.
These features help create visually appealing websites that work smoothly on desktops, tablets,
and mobile devices.
